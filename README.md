# Front-end Boilerplate
A front-end boilerplate for building sites or web apps


# Requirements

 * [node.js](https://nodejs.org/)
 * [npm](https://www.npmjs.com/)
 
# Dependencies

* Gulp
* Webpack 
* SASS
* Babel
* Eslint
* Stylelint

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
gulp default 
```

### Compiles and minifies for production
```
gulp build
```
